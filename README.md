## Getting Started

1. Clone Repository
```bash
git clone <your-repo-url>
cd folio
```

2. Install Dependencies
```bash
npm install
```

3. Run the development server:

```bash
npm run dev
```

4. Build for production:

```bash
npm run build
```

5. Preview production build:

```bash
npm run preview
```
